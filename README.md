# react-skylab
React application consuming REST API, project created in a Rocketseat course.

## Global dependencies
- NodeJs with Npm
- Yarn
- Package create-react-app

## Project dependencies
```js
  "dependencies": {
    "axios": "^0.19.0",
    "react": "^16.11.0",
    "react-dom": "^16.11.0",
    "react-router-dom": "^5.1.2",
    "react-scripts": "3.2.0"
  },
```

## Start server
1) `npm start`, in the project to start.
